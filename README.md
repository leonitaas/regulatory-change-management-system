# IRCMS - Intelligent Regulatory Change Management System

IRCMS is a multi-agent AI system designed to automate regulatory change management workflows.

## Project Structure

- `agents/` - AI agents and orchestration logic
- `data/` - Input and output files
- `schemas/` - Shared JSON/data schemas
- `policies/` - Configuration and policy files
- `utils/` - Helper functions and common utilities

## Team Workflow

- `main` branch contains stable code.
- Development is done in feature branches.
- Pull requests are used to merge changes into `main`.
